\doxysubsection{Detailed Description}
Core loop transformation functionality. 

\char`\"{}level\char`\"{} (starting from 1) means loop level and it corresponds to \char`\"{}dim\char`\"{} (starting from 0) in transformed iteration space \mbox{[}c\+\_\+1,l\+\_\+1,c\+\_\+2,l\+\_\+2,...., c\+\_\+n,l\+\_\+n,c\+\_\+(n+1)\mbox{]}, e.\+g., l\+\_\+2 is loop level 2 in generated code, dim 3 in transformed iteration space, and variable 4 in Omega relation. All c\textquotesingle{}s are constant numbers only and they will not show up as actual loops.

Formula\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{cp__als__nd__coo_8h_a61ffa457d87a5588b86d8e13aa34e420}{dim}} = 2*level -\/ 1}
\DoxyCodeLine{\mbox{\hyperlink{oc__quick__kill_8cc_aa7b9d53756ba6d974e0e786afaf1e6c6}{var}} = \mbox{\hyperlink{cp__als__nd__coo_8h_a61ffa457d87a5588b86d8e13aa34e420}{dim}} + 1}
\end{DoxyCode}
