\doxysubsection{Detailed Description}
\begin{DoxyVerb}A filter node selects a subset of the statement instances introduced by outer domain, expansion or extension
   nodes and retained by outer filter nodes. Filter nodes are typically used as children of set and sequence nodes,
   where the siblings select the other statement instances. Filters can also be used to select the statement
   instances that are mapped to a given value of a symbolic con- stant introduced in an outer context node.\end{DoxyVerb}
