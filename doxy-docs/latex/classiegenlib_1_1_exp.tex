\doxysubsection{Detailed Description}
An affine expression that allows uninterpreted function call terms. 

Memory management\+: the \mbox{\hyperlink{classiegenlib_1_1_exp}{Exp}} manages its own copies of any terms it contains, copies those when the \mbox{\hyperlink{classiegenlib_1_1_exp}{Exp}} itself is copied, and deletes them when the \mbox{\hyperlink{classiegenlib_1_1_exp}{Exp}} is destroyed.
