\doxysubsection{Detailed Description}
This class can store U\+Fcalls and their equ. \mbox{\hyperlink{classiegenlib_1_1_var_term}{Var\+Term}}. We need this in functionality of creating affine sets out of non-\/affine sets, where U\+Fcalls get replaced with symbolic constants. U\+F\+Call\+Terms and Var\+Terms are stored with coefficient = 1, however there is no need to set coefficient when inserting a \mbox{\hyperlink{classiegenlib_1_1_u_f_call_term}{U\+F\+Call\+Term}} or searching for \mbox{\hyperlink{classiegenlib_1_1_u_f_call_term}{U\+F\+Call\+Term}} or \mbox{\hyperlink{classiegenlib_1_1_var_term}{Var\+Term}} in the map using find functions.
