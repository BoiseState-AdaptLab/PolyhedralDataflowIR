For each computation node, compute the amount of data read (loaded), written (stored), the number of input and output streams, the number of int ops (I\+O\+Ps), floating point ops (F\+L\+O\+Ps). This leads to an estimate for the amount of memory traffic (Q) and total work (W). Arithmetic intensity can be computed as I = W/Q, the x-\/axis of the Roofline plot.
