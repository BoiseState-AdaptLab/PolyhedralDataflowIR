\doxysubsection{Detailed Description}
Data dependence vector and graph. 

All dependence vectors are normalized, i.\+e., the first non-\/zero distance must be positve. Thus the correct dependence meaning can be given based on source/destination pair\textquotesingle{}s read/write type. Suppose for a dependence vector 1, 0$\sim$5, -\/3), we want to permute the first and the second dimension, the result would be two dependence vectors (0, 1, -\/3) and (1$\sim$5, 1, -\/3). All operations on dependence vectors are non-\/destructive, i.\+e., new dependence vectors are returned.
