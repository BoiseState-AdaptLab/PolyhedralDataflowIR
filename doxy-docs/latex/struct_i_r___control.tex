\doxysubsection{Detailed Description}
Base abstract class for code structures. 

This is a place holder for the actual structure in the IR code. However, in cases that original source code may be transformed during loop initialization such as converting a while loop to a for loop or reconstructing the loop from low level IR code, the helper loop class (N\+OT I\+M\+P\+L\+E\+M\+E\+N\+T\+ED) must contain the transformed code that needs to be freed when out of service.
