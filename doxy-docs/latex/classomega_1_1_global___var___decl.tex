\doxysubsection{Detailed Description}
Abstract data type for Global Variable declarations. 

Supports the following operations\+:
\begin{DoxyItemize}
\item create global variable,
\item find the arity of the variable, (default = 0, for symbolic consts)
\item get the name of global variable,
\item tell if two variables are the same (if they are the same object) 
\end{DoxyItemize}
