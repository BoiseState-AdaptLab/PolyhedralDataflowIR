\doxysubsection{Detailed Description}
\begin{DoxyVerb}A domain node appears as the root of a schedule tree and introduces the statement instances that are scheduled
   by the descendants of the domain node. The statement instances are described by a named Presburger set.\end{DoxyVerb}
