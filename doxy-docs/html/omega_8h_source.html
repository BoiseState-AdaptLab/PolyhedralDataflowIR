37  *
38  * In many files, you can include just some of the following,
39  * specifically: if you are building a presburger tree, just
40  * include "pres_tree.h"; if you are querying it, include
41  * "pres_dnf.d" and "pres_conj.h"; if you are doing relational
42  * operations, include "Relation.h"
43  *
44  * Most of the function definitions are in the .c files with
45  * the same name as the .h that declares them, except:
46  * the remap and push_exists functions are in pres_var.c
47  * the DNFize functions are in pres_dnf.c
48  * the functions involving printing are in pres_print.c
49  * the beautify functions are in pres_beaut.c
