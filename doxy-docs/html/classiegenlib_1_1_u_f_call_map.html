Detailed Description

This class can store UFcalls and their equ. VarTerm. We need this in functionality of creating affine sets out of non-affine sets, where UFcalls get replaced with symbolic constants. UFCallTerms and VarTerms are stored with coefficient = 1, however there is no need to set coefficient when inserting a UFCallTerm or searching for UFCallTerm or VarTerm in the map using find functions.
