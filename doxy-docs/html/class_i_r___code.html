◆ FindArrayRef()

virtual std::vector<IR_ArrayRef *> IR_Code::FindArrayRef ( const omega::CG_outputRepr *  repr) const
pure virtual

Array references should be returned in their accessing order.

e.g. s1: A[i] = A[i-1]
s2: B[C[i]] = D[i] + E[i]
return A[i-1], A[i], D[i], E[i], C[i], B[C[i]] in this order.
