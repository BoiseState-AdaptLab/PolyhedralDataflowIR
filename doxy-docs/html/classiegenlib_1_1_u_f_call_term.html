Detailed Description

Represents a coefficient multiplied by an uninterpreted function call.

std::string  funcName,
unsigned int  num_args,
int  tuple_loc = -1 
)

Constructor

Memory management: this object assumes ownership of the passed-in Exp objects. The caller must not destroy them, and should not assume they survive destruction of this UFCallTerm.

Parameters
coeff– coefficient for the call
funcName– function to call
num_args– number of arguments the function call will have
args– arguments for the function call (adopted)
tuple_loc– index value if function returns a tuple and is being indexed, so term is still of type integer. If value is -1 assuming not in use.

If a tuple_loc is not specified and the function returns a tuple, then the UFCallTerm type is a tuple. size() will indicate size of tuple being returned.
