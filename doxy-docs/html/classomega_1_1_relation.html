Detailed Description

Relation representative.

Body and representative are separated to do reference counting to optimize copy of formulas. This owns a Rel_Body that contains the actual formula. Contains a lot of pipe that calls the corresponding functions in Rel_Body.

Could be a "set" or "relation",
